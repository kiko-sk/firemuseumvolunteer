# Fire Museum Volunteer App

消防博物馆志愿者管理系统

## 项目结构

```
2-firemuseumvolunteer/
├── gift-backend/          # 后端服务 (Node.js + Express)
├── volunteer-app-new/     # 前端应用 (React Native + Expo)
└── README.md
```

## 技术栈

### 前端
- React Native
- Expo
- React Navigation
- AsyncStorage

### 后端
- Node.js
- Express
- MongoDB (可选)

## 快速开始

### 1. 启动后端服务
```bash
cd gift-backend
npm install
npm start
```

### 2. 启动前端应用
```bash
cd volunteer-app-new
npm install
npx expo start
```

### 3. 测试应用
- 使用 Expo Go 应用扫描二维码
- 或在模拟器中运行

## 功能特性

- 志愿者注册/登录
- 活动管理
- 签到功能
- 个人中心
- 消息通知

## 开发环境

- Node.js >= 16
- Expo CLI
- Expo Go App (手机端测试)
